CREATING YOUR FIRST WEBPAGES

BEFORE

  • Before creating a webpage, there is one important step that must take place in order to be successful.
  • Paper and pen is all you need to complete this step.Drawing it out will reveal your vision.
  • Drawing it out will truly help bring your vision to life in a beautiful way.

    • INITIAL STEPS

    1. Create a folder
    2. Open folder in VS Code
    3. Boilerplate
    4. Break down into individual section
    5. Create each section in HTML
    6. Style each section using CSS
    7. Review your color scheme

    Note:

  • "Periodically" save your work as you go.
